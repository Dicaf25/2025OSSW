2학년 2학기 최대의 산.....
그 이름하여 고급자료구조라는 결코 피할수없는 이름의 산에서 9월 xx일 부터 12월 xx일까지의 생존여정이다....
산에서 생존하기 위한 그들의 고군분투 때로는 울고 때로는 절망하며 희망의 끝낯을 보려는 이들의 여정을 약 90일간의 생존일기로 서술해놓은 것을 드라마화 하였다.
드라마 <고급자료구조에서 살아남기>는 생존자들의 과거회상으로 시작하며 점차... 생존자들이 줄어가며 안타깝게도 중간에 좋지않은 결말로 가는 인원들도 존재하며, 최종 생존인원들이 어떻게 이 산에서 살아남는지, 구출되는 과정을 담은 드라마이다.. 드라마는 창원대 특히 컴퓨터 공학과 학생에게는 큰 충격을 안겨줄수도있으니 시청을 자제하는 편이 좋을것이다.... GOOD LUCK(당신에게도 행운이 깃들기를 진심으로 바란다)
